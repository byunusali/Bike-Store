Designed and implemented a retail business intelligence database system for a bike store using normalized relational database design principles in Microsoft SQL Server.

Imported and integrated multiple CSV datasets using BULK INSERT and advanced data transformation techniques in SQL to ensure accurate and structured data ingestion.

Developed analytical views, stored procedures, and aggregation queries to generate business insights including store revenue trends, top-selling products, and inventory availability.

Implemented key retail performance indicators (KPIs) such as Total Revenue, Average Order Value (AOV), inventory turnover rate, and staff sales performance metrics.

Automated data loading and reporting workflows using SQL Server Agent to schedule routine data processing and reporting tasks.

Applied data modeling and database optimization techniques to enhance query performance and support scalable retail analytics.
